Full list of key commands

General scope

Turn Flogic on or off. shift/
Increase tempo by 10 option =
Increase tempo by 1 option shift =
Decrease tempo by 10 option -
Decrease tempo by 1 option shift + -

Speak playhead position in bars beats and ticks. command f1
Speak playhead position in hours minutes and seconds command f4

Speak left locator command f2
Speak right locator command + f3

Speak division. Command + f6
Change division via the menu. Double press command f6
Turn replace on or off. ctrl shift r

track scope

Speak the name of the selected track and activate Track scope t

Volume up command up arrow
Volume up by 1 option command up arrow
Volume down command down arrow
Volume down by 1 option command down arrow
Speak volume v
Set volume to unity gain double press v

Pan left command left arrow
Pan left By 1 option command left arrow
Pan right command right arrow
Pan right by 1 option command right arrow
Speak pan p
Move pan to centre double press p

Speak current input for audio track I
Select input for audio track double press I
Speak output o
Select output double press o

Speak clip level l
Reset clip level double press l

Speak selected region. e
Select and speak next region. Right arrow
Select and speak previous region. Left arrow

Speak region gain g
Reset region gain double press g
Increase region gain by 3db ctrl command up arrow
Increase region gain by 1db shift ctrl command up arrow
decrease region gain by 3db ctrl command down arrow
decrease region gain by 1db Shift ctrl command down arrow

Speak gain reduction command g

Plugin scope

speak plugin in the selected slot and activate Plugin scope number keys 1 to 0
Open the plugin in the selected slot o
Speak the bypass state of the selected plugin b
Change the bypass state of the selected plugin. Double press b
Change the plugin in the selected slot c
Add a plugin to the end of the chain a
Insert a plugin between 2 adjacent plugins or at the start of the effects chain I
